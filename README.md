# KnC Automations Website Package

This package contains a cleaned, static website build for **KnC Automations**.

## Included
- Updated core website pages
- SEO-ready metadata, sitemap, robots, and contact page
- Field Nation bio text
- Rebrand checklist and name-change instructions

## Suggested deployment steps
1. Back up your current live site.
2. Upload these files to your hosting account.
3. Keep your current domain live while the new brand is rolled out.
4. Test the home page, services page, contact page, and sitemap.
5. Update Google Search Console and Google Business Profile after the new name is live.

## Notes
- This package intentionally removes the Git repository data from the export.
- The domain can remain unchanged while the branding shifts to KnC Automations.
